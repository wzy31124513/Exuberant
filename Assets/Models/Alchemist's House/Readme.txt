This is a set of lowpoly 3d objects that will allow you to create an interior location for your game. It can be used for top-down and first person view.

This pack contains over 90 prefabs:

- Walls (including round);
- Floor and ceiling;
- Stairs and pillars;
- Windows;
- Different furniture;
- Many kinds of props (books, scrolls, candles, laboratory equipment, vases etc.)
- Example Scene 

All models have hand painted textures, normal and specular maps
